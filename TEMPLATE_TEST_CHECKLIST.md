## Test Procedure

For each template, verify:

1. **2D Rendering**
   - [ ] Template displays correctly in 2D canvas
   - [ ] Animation works (if applicable)
   - [ ] Dimensions display correctly
   - [ ] Profile type renders correctly (square/chamfer/ovolo/round)
   - [ ] Glass panel renders (if enabled)
   - [ ] Frame color applies correctly

2. **3D Rendering**
   - [ ] 3D preview loads without errors
   - [ ] Model displays correctly
   - [ ] Animation works (if applicable)
   - [ ] Profile type visible in 3D
   - [ ] Glass transparency works

3. **Functionality**
   - [ ] Dimensions can be changed
   - [ ] Material selection works
   - [ ] Color selection works
   - [ ] Hardware options work
   - [ ] Pricing calculates correctly
   - [ ] Material breakdown shows correct values

4. **Mobile Compatibility**
   - [ ] Works on iPhone
   - [ ] Works on Android
   - [ ] Touch interactions work
   - [ ] Scrolling works in sidebar
