# ⌨️ Keyboard Shortcuts Guide

## Designer Shortcuts

### Undo/Redo
- **Ctrl+Z** (Windows/Linux) or **Cmd+Z** (Mac): Undo last change
- **Ctrl+Y** (Windows/Linux) or **Cmd+Shift+Z** (Mac): Redo last undone change

### Design Actions (Coming Soon)
- **Ctrl+S**: Save design
- **Ctrl+E**: Export design
- **Ctrl+P**: Calculate price
- **Esc**: Close dialogs/menus

### Canvas Navigation
- **Space + Drag**: Pan canvas (coming soon)
- **Scroll**: Zoom canvas (coming soon)

### 3D Preview
- **Left Click + Drag**: Rotate view
- **Scroll Wheel**: Zoom in/out
- **Right Click + Drag**: Pan view

---

## Global Shortcuts

### Navigation
- **Alt+H**: Go to Home (coming soon)
- **Alt+D**: Go to Designer (coming soon)
- **Alt+B**: Go to Dashboard (coming soon)

---

## Tips

- Most shortcuts work when focus is on the canvas/designer area
- Browser shortcuts (like Ctrl+T for new tab) take priority
- Some shortcuts may be disabled in production for security
