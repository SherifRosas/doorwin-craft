## â“ Troubleshooting

### **Problem: 3D preview is black/dark**
- **Solution**: This was fixed! It should now be light gray.

### **Problem: Can't rotate/drag doesn't work**
- **Solution**: Make sure you're clicking **inside the 3D preview area** (the gray box at bottom)
- Make sure you're using **left mouse button** (not right button)
- Try refreshing the page

### **Problem: Design doesn't show in 3D**
- **Solution**: Check that:
  1. Dimensions are set (width and height > 0)
  2. Glass is enabled (if you want to see glass)
  3. Try changing template or dimensions to refresh

### **Problem: 3D view is too close or too far**
- **Solution**: 
  1. Scroll to zoom out if too close
  2. Scroll to zoom in if too far
  3. Use right-click + drag to pan and adjust position
